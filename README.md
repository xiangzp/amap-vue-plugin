# demo

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

### Start
Register AMap plugins in main.js
```
Vue.use(vAMapPlugin, {
  v: '1.4.10',
  key: 'YOU_TOKEN'
})
```
